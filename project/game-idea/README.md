## 最重要の結論(1行版)

**1000万本は「面白いゲーム」が達成するのではなく、「面白さの種類・拡散の構造・生存の体質を同時に企画したゲーム」が達成する。**

---

## 調査から確立した5つの柱

### 柱1: 売上は掛け算である(`analysis/02`)

```
販売本数 ≒ 品質 × 拡散係数 × 発見性 × 摩擦の低さ × 再訪性 × タイミング(運)
```

- 品質(面白さ)は「ゼロにすると全部ゼロになる必要条件」だが、桁を決めるのは他の因子。
- 証拠: 「圧倒的に好評」のPentimentは14.7万本。一方、協力対応はリリースの6%で販売本数の36%を占める。同じウィッシュリスト数でも売上は10〜20倍ばらつく。

### 柱2: 面白さは4層構造で、上の層ほど拡散装置を兼ねる(`analysis/01`)

```
第4層: 文化(ミーム・儀式・Wiki・MOD)      → ゲーム外での常時露出
第3層: 観戦(クリップ映え・実況相性)        → 無料のグローバル広告
第2層: 社会(協力・会話・裏切り・感情同期)   → 友達を買わせる構造
第1層: 個人(ループ・成長・創発・創造・習熟) → 品質の土台
```

- 全要素を持つゲームは存在しない。1000万本は「2〜4要素を極限まで研いだ結果」。
- 勝ちパターンは3類型に収斂:
  - **A. ソロ深化型**(Stardew, Terraria, Vampire Survivors, Balatro)— 第1層を異常な密度で。1人開発可能な唯一の型。ロングテールの積分で到達
  - **B. 社会創発型**(Among Us, Lethal Company, Phasmophobia, PEAK, R.E.P.O.)— 第2層×第3層の同時設計。近年の最短ルート
  - **C. 世界没入型**(Valheim, Palworld, Rust)— 第1層フルセット+協力+EA継続更新。中規模チーム向け
- **全類型に共通する唯一の要素は「創発(毎回違う)」**。乱数・物理・他人という乱数——いずれの形でも、リプレイ性・配信映え・話題の寿命のすべてを支える。

### 柱3: 拡散は企画段階で決まる(`research/04`, `analysis/02`)

- 「遊んで面白い」「観て面白い」「誘いたくなる」「語りたくなる」は**独立した4つの性質**で、それぞれ設計しないと生まれない。
- 近接ボイスチャット、物理事故、1文で言えるコンセプト、記号的ビジュアル——これらは発売後のマーケでは後付けできない。
- Steamの発見アルゴリズムはウィッシュリストの「総数」ではなく「増加速度」に反応する。点火装置は配信・口コミ=拡散設計そのもの。

### 柱4: 運は消せないが期待値は管理できる(`research/03`, `research/04`)

- 着火は確率事象(Among Usは2年間同接8人だった。Lethal Companyは発売初週同接611人)。
- 期待値の管理手段: **①打席数を増やす**(Zeekerssは20作目、PEAKは1ヶ月ジャム+20万ドル)、**②低burn rateで場に残る**、**③デモ/Next Festを当たり判定に使う**、**④発売日程の競合回避**。
- 中国語(簡体字)ローカライズは1000万本の実質的な必要条件(Steam最大言語、非対応はレコメンド降格+68%が非購入)。

### 柱5: 2026年の市場は「メタは健在、コピーは死ぬ」(`research/05`)

- 協力ホラーの直球模倣・純サバイバルクラフト・純Balatroフォロワーは飽和。突破率は年数本レベル。
- しかし上位メタ「協力×創発×配信映え×低価格」は2026年もメガヒットを量産中(R.E.P.O. 1000万、PEAK 1000万、Meccha Chameleon 1500万——開発期間2ヶ月)。
- 勝者の共通項: **「一言で言える新しい動詞」を1つだけ発明する**(登山、ペイント擬態)。ジャンルの皮は差し替え可能なパラメータ。
- 需給ギャップ: ①プレミアム協力(埋まりつつある)、②ログライト×異ジャンル混成、③cozy/ソロ対応(キーワード+675%、5年で最も明確なトレンド)。
- 環境: Steam同接42M(史上最高)・年2万本リリース・中央値収益$249。母集団は最大、埋没確率も最大。

## 次フェーズ(アイデア出し)への引き継ぎ事項

調査・分析に基づく、企画が満たすべき要件セット:

**必須要件(これを欠くと桁が落ちる)**
1. コアループが1文で言える(できれば「新しい動詞」を1つ含む)
2. 創発装置を持つ(乱数 / 物理 / 他人、最低1つ——できれば複数)
3. 30秒の切り抜きが成立する(文脈なしで笑える/驚ける絵)
4. 90%+の好評を取れる品質(必要条件としての面白さ)
5. 簡体字中国語を含む多言語対応
6. 発見される設計(カプセル映え・WL速度を作るローンチ計画)

**戦略選択(どの類型で行くか——次フェーズの最初の意思決定)**
| 選択肢 | 型 | 価格帯 | 体制 | 主リスク |
|---|---|---|---|---|
| B. 社会創発型 | 協力×創発×配信映え+新動詞1つ | $6〜10 | 極小・短期(1〜6ヶ月) | 着火の運(→打席数で管理) |
| A. ソロ深化型 | ループ密度×習熟+ロングテール | $5〜15 | 1〜数人・中期 | 到達に年数がかかる |
| C. 世界没入型 | サバイバル系×EA継続更新 | $20〜30 | 中規模・長期 | 資本集約・飽和進行中 |

市場データが最も支持するのは**B**(前例が2023→2026年に毎年発生、短期開発で打席数を稼げる、需給ギャップ最大)。ただしBは「新しい動詞の発明」が全てであり、次フェーズの中心課題はここになる。

**Bで行く場合のアイデア出しの問い**
- まだ協力ゲームに持ち込まれていない「動詞」は何か?(登った、隠れた、拾った、撮った、は使用済み)
- その動詞は「悲鳴・笑い・裏切り・事故」のどれを量産するか?
- cozy/ソロ対応の間口(ソロでも遊べる設計)を両立できるか?

**未検証の論点(必要になったら追加調査)**
- 個別アイデアが決まった後の競合調査(同コンセプトの先行作の有無)
- 技術選定(近接ボイスチャット実装、ネットコードのコスト)
- 日本発であることの活かし方(Meccha Chameleonは日本の個人開発)
