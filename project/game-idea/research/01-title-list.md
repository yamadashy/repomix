# Steam 累計1000万本以上ゲーム 調査レポート

> 調査担当: リサーチエージェント(Opus) / 調査日: 2026-07-09

## 調査上の注記

- **「販売本数」の定義に注意**: 公式発表はほぼ全プラットフォーム合算(Steam + PS/Xbox 等)である一方、SteamSpy / Gamalytic / VG Insights / Alinea 等の推定は Steam 単体であることが多い。区分を各行に明記した。
- **公式値 vs 推定値**: 公式(開発元・パブリッシャー発表)は「公式」、第三者推定は「推定」と表記。
- **基本無料(F2P)タイトル**は「販売本数」が存在しないため、別表で「登録プレイヤー数/所有者数」として扱う(単純比較不可)。
- 数値は 2024〜2026年時点の最新確認値。調査時点で最も新しい報道を採用。
